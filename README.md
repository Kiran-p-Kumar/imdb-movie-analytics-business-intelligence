# IMDb Movie Analytics & Business Intelligence

## 📌 Overview
This project performs end-to-end data analysis on IMDb dataset to extract insights on movie trends, genre performance, ratings distribution, and content patterns.

## 🎯 Objectives
- Analyze genre dominance and audience preferences
- Study rating distribution across content
- Identify year-wise industry trends
- Provide data-driven insights for content strategy

## 🛠 Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI

## 🔄 Workflow
1. Data Cleaning (missing values, duplicates)
2. Exploratory Data Analysis (EDA)
3. Visualization (charts & insights)
4. Dashboard Creation (Power BI)

## 📊 Key Insights
- Certain genres dominate production volume
- Ratings are concentrated in mid-range values
- Content production shows trend growth over years
- Popular actors influence content visibility
